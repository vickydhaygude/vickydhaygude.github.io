01 / Enterprise AI · Capgemini

GxP AI Compliance System

LangChainLLMsPythonHuman-in-Loop

Problem

In regulated pharma/life sciences industries, every document change must be evaluated for GxP relevance. This was done manually — slow, inconsistent, and prone to compliance risk.

What I Built

An end-to-end LLM-based workflow that:

• Classifies document changes as GxP-relevant or non-GxP using a fine-tuned LLM
• Automatically generates audit log entries for every classification decision
• Produces URS (User Requirement Specification) documents from approved changes
• Routes all decisions through a human-in-the-loop approval step before finalising

Impact

Significantly reduced the time needed per compliance cycle. Full audit trail built-in. Deployed in an enterprise environment with regulated document workflows.

02 / Document Intelligence · Capgemini

AI Document Structuring Tool

PythonLLMsPDF ParsingDOCX

Problem

Enterprise document libraries contain thousands of PDFs and DOCX files with varying formats, layouts, and structures — making it impossible to search, process, or extract data reliably.

What I Built

A hybrid two-stage pipeline:

• Stage 1 (Rule-based): Extracts document structure — headings, sections, tables, lists — using layout and formatting rules
• Stage 2 (LLM): Intelligently restructures and cleans the extracted content into a consistent, structured format
• Output: Clean, uniform documents ready for downstream processing or search indexing

Impact

Eliminated hours of manual document cleanup per week. Made previously unprocessable documents available for downstream AI workflows.

03 / GenAI Chatbot · Capgemini

RAG Document Q&A Chatbot

LangGraphLangChainRAGStreamlit

Problem

Teams needed to extract answers from large, dense document sets quickly. Manual searching was slow and keyword search missed context-dependent answers.

What I Built

A full RAG (Retrieval-Augmented Generation) chatbot:

• LangGraph orchestrates the multi-step retrieval and generation workflow
• Documents are chunked, embedded, and stored for semantic search
• User queries trigger retrieval of the most relevant chunks
• LLM generates grounded, context-aware answers from retrieved content
• Streamlit provides a clean, simple UI for document upload and Q&A

Impact

Enabled multi-turn, context-aware Q&A on large document corpora. Reduced time to find information from minutes to seconds.
